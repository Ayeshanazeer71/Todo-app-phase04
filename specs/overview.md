# Overview: Phase II Todo Web App

## Project Purpose
Phase II transitions the Todo application from a standalone console tool to a full-stack web application with user authentication, persistent storage, and a modern UI.

## Current Phase: Phase II
Focus: Transition to full-stack web (FastAPI + Next.js).

## Tech Stack
- **Frontend**: Next.js 16+ (App Router), TypeScript, Tailwind CSS, Better Auth.
- **Backend**: FastAPI, SQLModel.
- **Database**: Neon PostgreSQL.
- **Authentication**: JWT-based (Better Auth integration).

## High-Level Features
1. User Authentication (Signup/Login) via Better Auth.
2. User-specific Task CRUD (Create, Read, Update, Delete).
3. Task Completion Toggling.
4. Persistent relational storage.
5. Filterable Task List.
